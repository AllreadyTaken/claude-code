# ADR-0020: MCP Integration

**Status:** Accepted
**Date:** 2026-04-01
**Domain:** Infrastructure
**Context:** `src/services/mcp/`, `@modelcontextprotocol/sdk`

## Context

External tools and data sources should be accessible through a standardized protocol.

## Decision

Integrate the Model Context Protocol (MCP) SDK to connect to external MCP servers for extended tool capabilities.

## Consequences

### Positive
- Standardized external tool integration
- Extensible tool ecosystem
- Decoupled tool servers

### Negative
- Network dependency for external tools
- Protocol versioning concerns
- Additional error handling

# ADR-0021: Analytics and Telemetry

**Status:** Accepted
**Date:** 2026-04-01
**Domain:** Infrastructure
**Context:** `src/services/analytics/`, `@opentelemetry/api`, GrowthBook

## Context

Product improvement requires usage analytics, feature flag evaluation, and telemetry.

## Decision

Use:
- **OpenTelemetry** for metrics and tracing
- **GrowthBook** for feature flag evaluation and analytics
- Custom analytics service for usage tracking

## Consequences

### Positive
- Data-driven product decisions
- Runtime feature flag control
- Performance monitoring

### Negative
- Privacy considerations
- Telemetry overhead
- Analytics data management
